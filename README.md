# 🎯 Enterprise AI Customer Sentiment Analytics Engine & Review Miner

A business intelligence and customer experience (CX) analytics workspace built using **Python**, the **Agno AI Framework**, **Pandas**, and **Plotly**, powered by **Google Gemini 2.5 Flash** models and wrapped in a responsive **Streamlit** architecture.

This engine automates text mining loops by transforming loose, conversational, and unstructured customer feedback logs into standardized relational data layers, parsing emotional weights, and delivering tactical data insights.

## ✨ Core Engineering Features
* **Unstructured Text Tokenization:** Ingests raw conversational review strings and isolates individual semantic nodes.
* **Quantitative Sentiment Scaling:** Utilizes specialized agent telemetry instructions to map feedback onto a strict mathematical spectrum ranging from -1.0 (Critical Threat) to +1.0 (Brand Prominence).
* **Multi-Dimensional Operational Tagging:** Automatically clusters reviews into operational enterprise categories (e.g., Shipping Logistics, Product Quality, Technical Stack Flaws).
* **Statistical Distribution Mapping:** Integrates native Plotly Express modules to compute dynamic bar visualization metrics showing brand sentiment distributions.

## 🛠️ System Engineering Stack
* **Agent Intelligence Tier:** Agno Framework
* **Core Language Model Core:** Google Gemini 2.5 Flash API
* **Mathematical Data Management:** Pandas, NumPy
* **Data Visualization Layers:** Plotly Express
* **Interface Architecture:** Streamlit Layout Engine
